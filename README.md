# AI-Powered Fitness Tracker

A full-stack fitness tracking application built using React, TypeScript, Strapi, and SQLite. The platform enables users to monitor their daily calorie intake, track physical activities, calculate BMI, manage fitness goals, and maintain a personalized fitness profile.

## Features

* User Authentication and Secure Login
* Personalized User Profile Management
* Food Logging and Daily Calorie Tracking
* Activity Logging and Calories Burned Monitoring
* BMI Calculation and Fitness Goal Tracking
* AI Food Snap using Google Gemini API
* Automatic Food Identification from Images
* AI-Based Calorie Estimation from Uploaded Food Images
* Dark and Light Theme Support
* Responsive User Interface
* REST API Integration with Strapi CMS

## Tech Stack

* Google Gemini API

## How It Works

1. Users create an account and complete onboarding by providing age, height, weight, and fitness goals.
2. Food intake and workout activities can be logged daily.
3. The application calculates BMI and tracks calorie consumption and expenditure.
4. Users can upload food images through the AI Food Snap feature.
5. Google Gemini analyzes the image and returns the detected food item along with estimated calorie information.
6. All user data is securely stored and managed through Strapi and SQLite.
